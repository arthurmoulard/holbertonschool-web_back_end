# ES6 Data Manipulation

## Description

Ce projet explore les structures de données et les méthodes de manipulation de tableaux introduites avec ES6 en JavaScript. Il couvre l'utilisation de `map`, `filter` et `reduce` sur les tableaux, les Typed Arrays, ainsi que les structures de données `Set`, `Map` et `WeakMap`.

## Objectifs d'apprentissage

À la fin de ce projet, vous devez être capable d'expliquer, sans l'aide de Google :

- Comment utiliser `map`, `filter` et `reduce` sur des tableaux
- Les Typed Arrays
- Les structures de données `Set`, `Map` et `WeakMap`

## Prérequis

- Ubuntu 20.04 LTS
- Node.js `20.x.x`
- npm `9.x.x`

## Installation

```bash

```

## Utilisation

```bash

# Lancer les tests complets (tests + lint)
```

## Requirements

- Tous les fichiers doivent se terminer par une nouvelle ligne
- Le code doit utiliser l'extension `.js`
- Le code est testé avec **Jest** via `npm run test`
- Le code est vérifié avec **ESLint**
- Toutes les fonctions doivent être exportées
